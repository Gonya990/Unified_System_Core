# 🔍 ЧАСТЬ 1: ДИАГНОСТИКА AC СТОРОНЫ

## Расположение контактов на вилке

```
      ВИЛКА БП (вид спереди)
    ┌─────────────────┐
    │                 │
    │   L    N    ⏚   │
    │   ○    ○    ○   │
    │                 │
    └─────────────────┘
        │    │    │
        │    │    └── Земля (заземление)
        │    └─────── Ноль (нейтраль)
        └──────────── Фаза (линия)
```

## Проверка сопротивлений на AC входе

**Режим мультиметра: Ω (сопротивление)**

### Тест 1: Проверка щупов

```
Замкни щупы друг с другом
Результат: _______ Ом (запомни, это погрешность)
```

### Тест 2: N ↔ ⏚ (ноль — земля)

```
Щупы между N и ⏚
Ожидание: 0 - 100 Ом (норма из-за Y-конденсаторов)
Результат: _______ Ом
```

### Тест 3: L ↔ ⏚ (фаза — земля) ⚠️ ВАЖНО

```
Щупы между L и ⏚
Ожидание: МОмы (мегаомы) или OL (бесконечность)
Результат: _______ Ом

⚠️ Если < 1 Ом = КОРОТКОЕ ЗАМЫКАНИЕ!
```

### Тест 4: L ↔ N (фаза — ноль)

```
Щупы между L и N
Ожидание: 50+ Ом, постепенно растёт (конденсаторы заряжаются)
Результат: _______ Ом
```

## Таблица результатов AC диагностики

| Измерение | Норма | Проблема | Твой результат |
|-----------|-------|----------|----------------|
| N ↔ ⏚ | 0-100 Ом | >1 кОм подозрительно | 0.05 Ом |
| **L ↔ ⏚** | **МОмы/OL** | **<1 Ом = КЗ!** | **0.05 Ом ❌** |
| L ↔ N | 50+ Ом ↑ | <1 Ом = КЗ | 50 Ом ✅ |

## Как найти +5V Standby

> ⚡ **ВНИМАНИЕ:** Для этого теста БП должен быть включён в сеть!
>
> ⚠️ **НО!** У тебя КЗ — НЕ ВКЛЮЧАЙ БП до ремонта!

```
Режим: DCV 20V
Чёрный щуп: на GND (найденный)
Красный щуп: тыкай другие пины

~5V = +5VSB (дежурное питание)
~12V = +12V линия
~3.3V = +3.3V линия
```

## Проверка компонентов мультиметром

### Тест MOV

```
1. Найди MOV (диск у входа AC)
2. Мультиметр → режим Ω или прозвонки
3. Измерь между ножками MOV:

   Результат: _______ Ом
   
   МОмы или OL → ✅ MOV исправен
   0-несколько Ом → ❌ MOV ПРОБИТ!
```

### Тест Y-конденсатора

```
1. Найди Y-cap между L и ⏚
2. Мультиметр → режим Ω
3. Измерь на его выводах:

   Результат: _______ Ом
   
   МОмы или OL → ✅ Конденсатор OK
   0-несколько Ом → ❌ ПРОБИТ!
```

### Метод изоляции

```
Если визуально не видно — изолируй по одному:

1. ВЫКЛЮЧИ БП из сети!
2. Выпаяй или откуси ОДНУ ножку подозреваемого
3. Измерь снова L ↔ ⏚ на вилке
4. Если КЗ исчезло (МОмы) → виновник найден! ✅
5. Если КЗ осталось → ищи дальше

⚠️ Начни с MOV — это самый частый виновник!
```

## Вариант А: Пробитый MOV

### Способ 1: Удаление (быстро)

```
┌───────────────────────────────────────────────────┐
│                                                   │
│  MOV — это защита от скачков напряжения в сети.  │
│                                                   │
│  БП будет работать БЕЗ MOV, но не будет          │
│  защищён от скачков напряжения!                  │
│                                                   │
│  Действия:                                        │
│  1. Выпаяй MOV или откуси обе ножки              │
│  2. Проверь L ↔ ⏚ — должно быть МОмы            │
│  3. Собери БП                                    │
│  4. Проверь работу                               │
│                                                   │
└───────────────────────────────────────────────────┘
```

### Способ 2: Замена (правильно)

```
1. Запиши маркировку старого MOV (например "14D471K")
2. Купи такой же:
   • Напряжение: 470V (для 230V сети)
   • Диаметр: такой же как был
3. Выпаяй старый
4. Впаяй новый (соблюдай полярность не нужно)
5. Проверь L ↔ ⏚
```

**Где купить:**

- Радиорынок
- AliExpress: "MOV 14D471" или "Varistor 470V"
- Магазины электроники

### Способ 1: Удаление (временно)

```
┌───────────────────────────────────────────────────┐
│                                                   │
│  Y-конденсатор фильтрует помехи.                 │
│                                                   │
│  БП будет работать БЕЗ него, но может            │
│  создавать помехи для других устройств.          │

│  Действия:                                        │
│  1. Выпаяй Y-cap или откуси ножки                │
│  2. Проверь L ↔ ⏚ — должно быть МОмы            │
│  3. Собери и проверь работу                      │
│                                                   │
└───────────────────────────────────────────────────┘
```

### Способ 2: Замена (правильно)

```
1. Запиши параметры:
   • Ёмкость (например 2200pF = 2.2nF)
   • Класс (Y1 или Y2)
   • Напряжение (обычно 250-400VAC)

2. Купи ТАКОЙ ЖЕ:
   ⚠️ ОБЯЗАТЕЛЬНО класс Y1 или Y2!
   ⚠️ Обычные конденсаторы НЕЛЬЗЯ!

3. Выпаяй старый, впаяй новый
```

**Где купить:**

- "Y2 capacitor 2.2nF 275VAC" на AliExpress
- Магазины радиодеталей

---

## После ремонта — проверка

```
1. Собери БП (закрути крышку)

2. Проверь ещё раз L ↔ ⏚:
   Должно быть: МОмы или OL ✅

3. Подключи к розетке через удлинитель
   с выключателем (для безопасности)

4. Включи — наблюдай:
   • Не должно искрить
   • Не должно выбивать пробки
   • Вентилятор может не крутиться (Zero-RPM норма)

5. Проверь +5V Standby на MB-разъёме

6. Замкни PS_ON + GND для полного запуска

7. Измерь все напряжения (+12V, +5V, +3.3V)
```
