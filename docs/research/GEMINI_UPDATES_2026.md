# Gemini Code Assist Updates (2025-2026)

Summary of key features and changes based on the latest release notes.

## 🚀 Key Features

### Agent Mode & MCP Support (Oct 2025)

* **Deprecation of Tools**: Fixed tools (@-symbols) were replaced by **Agent Mode**.
* **MCP Servers**: Native support for Model Context Protocol (MCP) to connect to external services.
* **Strategic Alignment**: This shift confirms the "Agentic" direction of the Unified System is industry-standard.

### Next Edit Predictions (Sept/Oct 2025)

* **IDE Support**: Available in both VS Code and IntelliJ.
* **Functionality**: Predicts the next logical code suggestion throughout the current file.

### Finish Changes (Dec 2025)

* **IntelliJ Feature**: AI pair programmer that observes in-progress work (pseudocode, #TODOs) and completes the job.

### Inline Diff (Nov 2025)

* **IntelliJ GA**: Chat-generated code suggestions are visually highlighted directly in the editor.

## 🛠 Recent Updates (Jan 2026)

* **VS Code v2.68.0**: Various bug fixes and minor product enhancements (Jan 23, 2026).

## ⚠️ Important Transitions

* **Code Customization**: Now supported in Agent Mode and Gemini CLI (Nov 12, 2025).
* **Persistent Memory**: Now supported on GitHub for context retention across interactions (Nov 10, 2025).
