## Core Strategy: "Proof of Authorship"

### 1. The Problem

- AI-generated code creates ambiguity in authorship.
- Grant committees require proof of human innovation and ownership.

### 2. The Solution: "Digital Ledger of Will"

- **Git as Evidence:** Use the Git commit history not just for version control, but as a legal log.
- **"Intent Ledger":** Maintain a file (like `INTENT_LEDGER.yaml`) that maps high-level human intent to low-level AI implementation. This proves the **idea** was yours, even if the **typing** was AI.
- **Biometric Signing:** explored in `secure_vault`, linking code commits to your biological identity (voice/face).
