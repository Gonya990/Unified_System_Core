// turbo-all

1. Pull latest changes:

```bash
git pull --rebase origin main
```

2. Stage all changes:

```bash
git add -A
```

3. Show status:

```bash
git status --short
```

4. Commit with appropriate message (group related changes logically)

5. Push to origin:

```bash
git push origin main
```
