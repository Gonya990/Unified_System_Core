# Bullet Point Formulas

## PAR (Problem-Action-Result)
```
[Identified problem/challenge], [action taken using specific tools/methods],
resulting in [quantified improvement/outcome].
```

## Result-First
```
[Achieved X% improvement/generated $X] by [action taken] through [method/tools used].
```

## STAR Condensed
```
[Action verb] [task/responsibility] in [context/situation], achieving [result with metrics].
```

## Action + Skill + Result
```
[Developed/Created/Implemented] [deliverable] using [technologies],
resulting in [measurable outcome].
```
