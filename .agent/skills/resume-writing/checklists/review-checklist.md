# Resume Review Checklist

Use this checklist before submitting any resume.

---

## Format & Appearance

- [ ] **Length appropriate** (1 page for <10 years exp, 2 pages for senior)
- [ ] **Consistent formatting** (fonts, spacing, bullet styles match throughout)
- [ ] **Professional font** (Arial, Calibri, Garamond, 10-12pt)
- [ ] **Adequate margins** (0.5-1 inch)
- [ ] **Clear section headers** (Experience, Education, Skills)
- [ ] **No graphics, tables, or columns** (ATS compatibility)
- [ ] **White space balanced** (not cramped, not sparse)
- [ ] **Contact info complete** (name, phone, email, LinkedIn, city/state)

---

## Content Quality

### Professional Summary (if included)
- [ ] 60-100 words maximum
- [ ] States professional identity clearly
- [ ] Includes 2-3 key achievements
- [ ] Relevant to target role
- [ ] Not generic/cookie-cutter

### Work Experience
- [ ] Most recent role has most bullets (5-6)
- [ ] Older roles have fewer bullets (3-4)
- [ ] Each bullet starts with action verb
- [ ] Achievements emphasized over duties
- [ ] Metrics/numbers included where possible
- [ ] Company context provided (size, industry)
- [ ] Consistent verb tense (present for current, past for previous)
- [ ] No first-person pronouns ("I", "my")
- [ ] Relevant experience highlighted first

### Skills Section
- [ ] Technical skills listed (not just soft skills)
- [ ] No Word/Excel unless specifically relevant
- [ ] Keywords from job description included
- [ ] Organized by category
- [ ] No self-assessment ratings (unless industry standard)

### Education
- [ ] Placed appropriately (top if recent grad, bottom if experienced)
- [ ] GPA included only if >3.5 (or required)
- [ ] Relevant coursework listed (if recent grad)
- [ ] Dates included

---

## Tailoring for This Application

- [ ] Reviewed job description thoroughly
- [ ] Identified 5+ keywords from posting
- [ ] Keywords incorporated naturally in bullets
- [ ] Most relevant experience appears first in each role
- [ ] Summary customized for this role (if included)
- [ ] Skills section matches job requirements
- [ ] Company/role language mirrored where appropriate

---

## ATS Optimization

- [ ] Standard section headers used
- [ ] No headers/footers with important info
- [ ] No tables or text boxes
- [ ] Saved as .docx (preferred) or .pdf (if specified)
- [ ] File named professionally: `FirstName_LastName_Resume.docx`
- [ ] No special characters that might break parsing

---

## Common Mistakes Check

- [ ] No typos or grammatical errors
- [ ] No "References available upon request"
- [ ] No objective statement (use summary instead)
- [ ] No photos (unless required in some countries)
- [ ] No personal info (age, marital status, etc.)
- [ ] No generic phrases ("responsible for", "duties included")
- [ ] No buzzwords without evidence ("team player", "hard worker")
- [ ] No unexplained gaps (addressed if significant)

---

## Quantification Check

Review each bullet point:

| Bullet | Has Number? | Could Add One? | Action Needed |
|--------|-------------|----------------|---------------|
| 1      | [ ] Yes / No | [ ] Yes / No | |
| 2      | [ ] Yes / No | [ ] Yes / No | |
| 3      | [ ] Yes / No | [ ] Yes / No | |
| 4      | [ ] Yes / No | [ ] Yes / No | |
| 5      | [ ] Yes / No | [ ] Yes / No | |

**Goal:** At least 60% of bullets should have quantifiable metrics.

---

## Action Verb Check

Ensure variety. Avoid repeating the same verb:

- [ ] "Managed" used maximum 2 times
- [ ] "Led" used maximum 2 times
- [ ] "Developed" used maximum 2 times
- [ ] Each section has different starting verbs

---

## Final Review

- [ ] Read aloud for flow and clarity
- [ ] Have someone else proofread
- [ ] Printed to check formatting
- [ ] All links work (LinkedIn, portfolio)
- [ ] Email address is professional
- [ ] Phone number is correct

---

## Before Sending

- [ ] Correct company name in all materials
- [ ] Correct job title referenced
- [ ] Cover letter matches resume (if sending both)
- [ ] Both documents have consistent formatting
- [ ] Email body is professional (not the cover letter)
- [ ] Attachments actually attached

---

## Post-Application

- [ ] Added to application tracking spreadsheet
- [ ] Noted which resume version sent
- [ ] Set calendar reminder for follow-up (1-2 weeks)
- [ ] Saved copy of job posting (in case it's removed)
