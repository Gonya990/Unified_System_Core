✅ What is Droid CLI / Factory AI

Factory AI is an “agent-native” development platform — its core idea is to let “droids” (AI agents) handle large parts of the software lifecycle.

🛠 What it can do (Capabilities)

Using Droid CLI / Factory AI, you can:

Analyze a codebase: ask the agent to explain architecture, dependencies, main entry points, test setups, etc.
