## Фаза 2: ADB over WiFi

### Шаг 2.1: Включить режим разработчика

**На планшете:**

1. Настройки → О телефоне → Номер сборки
2. Нажать 7 раз на "Номер сборки"
3. Ввести PIN/пароль
4. "Вы стали разработчиком"

### Шаг 2.2: Включить отладку

1. Настройки → Система → Для разработчиков
2. Включить "Отладка по USB"
3. Включить "Беспроводная отладка" (Android 11+) или использовать USB сначала

### Шаг 2.3: Подключение ADB

```bash
# На ПК (Windows/WSL или Mac)
# Сначала по USB:
adb devices
adb tcpip 5555
adb connect 192.168.190.111:5555

# Теперь USB можно отключить
# Проверка:
adb devices
# Должен показать: 192.168.190.111:5555 device
```

### Шаг 2.4: Установка scrcpy

```bash
# macOS
brew install scrcpy

# Windows (PowerShell)
scoop install scrcpy
# или скачать с https://github.com/Genymobile/scrcpy/releases

# Запуск зеркалирования:
scrcpy -s 192.168.190.111:5555
```

## Фаза 3: Tasker + AutoRemote

### Установка через ADB (если Play Store недоступен)

```bash
# Скачать APK и установить
adb -s 192.168.190.111:5555 install tasker.apk
adb -s 192.168.190.111:5555 install autoremote.apk
```

### Настройка AutoRemote для Home Assistant

1. Открыть AutoRemote на планшете
2. Зарегистрировать устройство (получить Personal URL)
3. В Home Assistant добавить notify сервис:

```yaml
# configuration.yaml
notify:
  - platform: rest
    name: arthur_tablet
    resource: https://autoremotejoaomgcd.appspot.com/sendmessage
    method: POST
    data:
      key: "YOUR_AUTOREMOTE_KEY"
      message: "{{ message }}"
```

### Примеры автоматизаций Tasker

| Триггер | Действие |
|---------|----------|
| Время 21:00 | Включить режим "Не беспокоить", уменьшить яркость |
| Приложение YouTube | Отправить уведомление родителям |
| Геолокация (ушёл из дома) | Уведомить родителей |
