## 🌐 Tailscale — Удалённый доступ отовсюду

| Параметр | Значение |
| -------- | -------- |
| **Tailscale IP** | `100.114.27.103` |
| **Hostname** | `lenovo-tb-j606f` |
| **ADB Port** | `5555` |

### Быстрые команды

```bash
# Подключиться к планшету (из любой точки мира через Tailscale)
adb connect 100.114.27.103:5555

# Проверить подключение
adb -s 100.114.27.103:5555 shell echo "Connected!"

# Зеркалирование экрана
scrcpy -s 100.114.27.103:5555 --window-title "Планшет Артура"

# Сделать скриншот
adb -s 100.114.27.103:5555 exec-out screencap -p > ~/Desktop/arthur_screen.png

# Установить APK
adb -s 100.114.27.103:5555 install /path/to/app.apk

# Открыть приложение
adb -s 100.114.27.103:5555 shell am start -n com.package.name/.MainActivity

# Геолокация (через dumpsys)
adb -s 100.114.27.103:5555 shell dumpsys location | grep -A5 "last location"
```

## 🔧 Полезные ADB команды


## ⚠️ Важные заметки

1. **Tailscale должен быть включен** на планшете для удалённого доступа
2. **USB-кабель не нужен** — доступ полностью беспроводной
3. **Работает из любой сети** — дома, на работе, в путешествии
4. **Family Link основной** — для ежедневного контроля используй приложение Family Link
5. **ADB для продвинутого контроля** — установка, скриншоты, управление

---

## 🚨 Если потеряли доступ

1. Проверь что планшет онлайн: `tailscale ping lenovo-tb-j606f`
2. Проверь статус Tailscale: `tailscale status | grep lenovo`
3. Если планшет оффлайн — дождись когда он подключится к интернету
4. Если ADB не отвечает — нужен физический доступ для переподключения
