### 🌐 Браузерная Автоматизация (Nodriver)

**Статус:** ✅ Внедрено

- **Решение:** Unix Socket Daemon + CLI клиент (`ndc`).
- **Преимущество:** Снижение потребления токенов в 5-6 раз по сравнению с MCP.
- **Возможности:** Полный доступ к профилю Chrome (пароли, расширения, сессии) без риска детекции ботов.

- **Безопасность:** Ключи зашифрованы и управляются через `config_manager`.
