## 📋 Стандартная Форма Регистрации Устройства

### **Анкета Нового Устройства:**

```yaml
device_registration:
  # Базовая информация
  tailscale_ip: "100.x.x.x"
  hostname: "device-name"
  local_ip: "192.168.x.x" (если есть)
  
  # Владелец
  owner:
    name: "Имя"
    telegram_id: 123456789
    telegram_username: "@username"
  
  # Назначение
  role: "DEVELOPER|ADMIN|FAMILY|etc"
  purpose: "Краткое описание назначения"
  projects_access:
    - "ai_core"
    - "content_factory"
    - "etc"
  
  # Права доступа
  permissions:
    read: true/false
    write: true/false
    execute: true/false
    delete: true/false
    admin: true/false
    manage_users: true/false
  
  # Технические характеристики
  hardware:
    type: "laptop|phone|server|tablet|etc"
    os: "macOS|Windows|Linux|iOS|Android"
    cpu: "описание"
    ram: "размер"
    storage: "размер"
  
  # Статус
  status: "online|offline|planned"
  deployment_date: "YYYY-MM-DD"
  notes: "Дополнительные примечания"
```
