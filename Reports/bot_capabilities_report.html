💡 Примечание для авторизованных пользователей

Пользователи без личных API ключей используют общий Gemini API администратора. Личные данные (здоровье, история чата) хранятся отдельно для каждого пользователя. Gmail и Calendar требуют отдельной OAuth авторизации для каждого пользователя.
