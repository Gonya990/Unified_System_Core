# Настройка Голосового Управления (Яндекс Алиса)

Этот документ описывает процесс настройки интеграции Unified AI Bot с Яндекс Алисой.

## Архитектура

1. **Пользователь** говорит команду Алисе ("Попроси Гоню...").
2. **Яндекс** преобразует голос в текст и отправляет JSON на Webhook.
3. **Cloudflare Tunnel** принимает запрос из интернета и передает его на локальный сервер `igor-gaming-1`.
4. **Unified AI Bot** (`alice_skill.py`) на порту 8090 обрабатывает запрос.
5. **AI** выполняет действие (включает свет, ищет инфу) и возвращает ответ текстом.
6. **Алиса** озвучивает ответ.

## Шаг 1: Подготовка Сервера

Убедитесь, что бот запущен и слушает порт 8090:

```bash
sudo systemctl status ai-bot
# В логах должно быть: "Alice Skill server started on port 8090"
```

## Шаг 2: Настройка Туннеля

На сервере (`igor-gaming-1`) установите и запустите `cloudflared`:

```bash
# Установка
curl -L --output cloudflared.deb https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64.deb
sudo dpkg -i cloudflared.deb

# Запуск туннеля (временный)
cloudflared tunnel --url http://localhost:8090
```

Скопируйте полученный URL (например `https://frosty-glacier-42.trycloudflare.com`).

> **Для постоянной работы:** Рекомендуется настроить именованный туннель Cloudflare Zero Trust, чтобы URL не менялся при перезапуске.

## Шаг 3: Настройка Навыка в Яндексе

1. Перейдите в [Консоль Разработчика Яндекс.Диалогов](https://dialogs.yandex.ru/developer).
2. Нажмите **"Создать диалог"** -> **"Навык в Алисе"**.
3. **Название**: "Гоня Ассистент" (или "Умный Дом").
4. **Webhook URL**: Вставьте ваш Cloudflare URL + `/alice`
   - Пример: `https://frosty-glacier-42.trycloudflare.com/alice`
5. **Голос**: Выберите голос (например, "Оксана" или "Джейн").
6. **Приватность**: Поставьте галочку "Не показывать в каталоге", если хотите использовать только сами.
   - *Совет:* Чтобы использовать приватный навык, нужно добавить свой аккаунт Яндекса в "Тестирование" или просто войти под тем же аккаунтом.

## Шаг 4: Тестирование

В консоли Яндекса откройте вкладку **"Тестирование"**.
Напишите: `Привет` или `Статус`.
Бот должен ответить.

## Шаг 5: Использование

Скажите Алисе (на колонке или в телефоне):

- **"Запусти навык Гоня Ассистент"** (для начала диалога)
- **"Алиса, попроси Гоню..."**

### 🗣 Доступные голосовые команды

#### 🏠 Умный Дом

- *"Включи свет в гостиной"*
- *"Выключи всё"*
- *"Какая температура в спальне?"*
- *"Закрой шторы"*

#### 🧠 Информация и Поиск

- *"Найди рецепт борща"* (Web Search)
- *"Что изображено на этом фото?"* (если контекст фото доступен)
- *"Расскажи про погоду"*

#### ✅ Задачи и Напоминания

- *"Напомни через 10 минут выключить духовку"*
- *"Добавь задачу купить молоко"*
- *"Что у меня в списке задач?"*

#### 📅 Календарь и Планирование (Google Calendar & Linear)

- *"Что у меня в календаре на завтра?"*
- *"Добавь встречу с врачом в пятницу в 14:00"*
- *"Создай задачу в Linear: пофиксить баг с авторизацией"*
- *"Прочитай задачи из Linear"*

#### ⚙️ Управление и Администрирование

- *"Статус системы"* (Инфраструктура)
- *"Сколько я потратил токенов?"*
- *"Запусти поиск вакансий"* (Job Hunter)
- *"Обнови систему"* (Запуск авто-обновления)
- *"Сделай бэкап базы данных"*

## Устранение неполадок

**Алиса говорит "Навык не отвечает":**

1. Проверьте, запущен ли `cloudflared`.
2. Проверьте, запущен ли бот (`sudo systemctl status ai-bot`).
3. Проверьте логи бота: `journalctl -u ai-bot -f`. Должны быть видны входящие запросы.
