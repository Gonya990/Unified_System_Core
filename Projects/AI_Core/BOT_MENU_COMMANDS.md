## 📋 КАК УСТАНОВИТЬ МЕНЮ

1. **Открой @BotFather в Telegram**
2. **Отправь команду:** `/setcommands`
3. **Выбери своего бота**
4. **Скопируй и вставь ВСЁ ниже:**

```
start - Main menu
help - Show help and available commands
settings - AI model & provider settings
models - List and switch AI models
setprovider - Switch AI provider
beads - Manage task tracking
agent - Run AI agent (code-explorer, reviewer...)
pipeline - Run agent pipeline (feature, bugfix...)
brief - Daily/weekly calendar brief
img - Generate image with DALL-E 3
generate_video - Generate video from prompt
video_status - Check video generation status
factory - AI Content Factory control
status - System status
infra - Infrastructure overview
mail - Check email
calendar - Show calendar events
todo - Task management
search - Web search
ha - Home Assistant control
aivoice - Generate speech with ElevenLabs
aisub - Generate stylized subtitles
imagine - Generate image (alias for /img)
usage - Usage statistics
costs - Cost tracking
clear - Clear conversation memory
backup - Backup data
health - Health check
```
