# План реализации (Implementation Plan)

## Описание цели

Спланировать и реализовать сложного Telegram-бота «Личный помощник», который интегрируется с Google Calendar, сохраняет долгосрочную память/контекст, активно взаимодействует с пользователем и предоставляет интеллектуальную помощь в повседневных задачах и бизнес-аналитике.

## Требуется проверка пользователем

> [!IMPORTANT]
> Бот требует учетные данные Google Cloud Project (OAuth 2.0 Client ID) для доступа к Calendar API. Пользователь должен авторизовать приложение.

> [!NOTE]
> Необходимо подтвердить выбор базы данных для «Памяти» и «Контекста». Предлагается SQLite для простоты, или мы можем использовать векторное хранилище, если под «аналитическими выводами» (digest insights) подразумевается семантический поиск.

## Предлагаемая логика архитектуры

### Компоненты

1. **Фронтенд бота (Telegram)**: `ai_telegram_bot_v2.py` (улучшенный).
2. **Интерфейс LLM**: Существующая интеграция с Ollama для понимания естественного языка (анализ намерений, генерация резюме).
3. **Сервис Google**: Модуль для обработки потока OAuth и вызовов Calendar API.
4. **База данных (База знаний)**:
    - `Users`: ID, токен аутентификации Google (зашифрованный/хранится безопасно), предпочтения.
    - `Events`: ID, CalendarEventID, описание контекста, дата создания.
    - `ChatMemory`: Векторный или сокращенный текст прошлых взаимодействий.
5. **Планировщик (Scheduler)**: Фоновый поток/процесс для сканирования предстоящих событий и «неактивных» пользователей для отправки уведомлений.

## План проверки

### Ручная проверка

1. **Онбординг**: Запустите бота, используйте `/start`, проверьте генерацию ссылки Google Auth.
2. **Календарь**: Попросите бота «Запланировать тестовую встречу», проверьте реальный Google Календарь.
3. **Контекст**: Спросите «О чем эта тестовая встреча?», проверьте, что бот вспоминает контекст, а не только название.
4. **Подталкивание (Nudge)**: Вручную запустите функцию «проверки неактивности» (или установите порог на 1 минуту) и убедитесь, что бот отправляет сообщение для вовлечения.
