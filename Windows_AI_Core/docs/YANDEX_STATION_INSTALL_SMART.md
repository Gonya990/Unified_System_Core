# Установка Yandex.Station на Home Assistant (smart - 192.168.1.216)

## Быстрая установка через веб-интерфейс

### Шаг 1: Откройте Home Assistant

Перейдите по адресу: **<http://192.168.1.216:8123>** (или <http://smart.tail-scale.ts.net:8123>)

### Шаг 2: Откройте HACS

1. В боковом меню найдите **HACS**
2. Если HACS не установлен, сначала установите его: <https://hacs.xyz/docs/setup/download>

### Шаг 3: Добавьте репозиторий

1. В HACS нажмите на **три точки** (⋮) в правом верхнем углу
2. Выберите **"Custom repositories"**
3. Введите:
   - Repository: `AlexxIT/YandexStation`
   - Category: `Integration`
4. Нажмите **"ADD"**

### Шаг 4: Установите интеграцию

1. Найдите в списке **"Yandex.Station"**
2. Нажмите на неё
3. Нажмите **"Download"** (или "Install")
4. Выберите последнюю версию
5. Нажмите **"Download"**

### Шаг 5: Перезагрузите HA

1. Настройки → Система → Перезагрузить
2. Подождите ~1-2 минуты

### Шаг 6: Добавьте интеграцию

1. Настройки → Устройства и службы
2. Нажмите **"+ Добавить интеграцию"** (внизу справа)
3. Найдите **"Yandex.Station"**
4. Выберите метод авторизации:

#### QR-код (Рекомендуется)

1. Отсканируйте QR-код приложением **Яндекс** на телефоне
2. Подтвердите вход
3. Готово!

#### Cookies (Альтернатива)

1. Откройте <https://yandex.ru> в браузере
2. Войдите в свой аккаунт
3. Нажмите F12 → Application → Cookies → <https://yandex.ru>
4. Найдите cookie `Session_id`
5. Скопируйте его значение
6. Вставьте в поле интеграции

### Шаг 7: Проверка

После успешной настройки:

1. Перейдите в **Настройки → Устройства и службы**
2. Вы должны увидеть **"Yandex.Station"** с вашей станцией
3. Станция появится как **Media Player** на главном экране

## Использование

### TTS (Голосовые сообщения)

Теперь можно отправлять голосовые сообщения на станцию:

**Через Developer Tools → Services:**

```yaml
service: tts.yandex_station_say
data:
  entity_id: media_player.yandex_station_duo_max
  message: "Привет! Это тестовое сообщение"
```

**Через автоматизацию:**

```yaml
automation:
  - alias: "Доброе утро"
    trigger:
      platform: time
      at: "08:00:00"
    action:
      service: tts.yandex_station_say
      data:
        entity_id: media_player.yandex_station_duo_max
        message: "Доброе утро! Время вставать"
```

### Zigbee устройства

Все Zigbee устройства, подключенные к станции, автоматически появятся в HA.

### Управление через AI Bot

После настройки можно управлять через наш Telegram бот:

```
/ha status
/ha sensors
```

## Troubleshooting

**Проблема:** HACS не установлен
**Решение:** Установите HACS: <https://hacs.xyz/docs/setup/download>

**Проблема:** Не могу найти Yandex.Station в интеграциях
**Решение:** Убедитесь, что:

1. Репозиторий добавлен в Custom repositories
2. Интеграция установлена через HACS
3. Home Assistant перезагружен

**Проблема:** Ошибка авторизации
**Решение:** Используйте QR-код вместо cookies

**Проблема:** Станция не появляется
**Решение:**

1. Проверьте, что станция в той же сети (192.168.1.x)
2. Убедитесь, что станция включена
3. Перезагрузите станцию
