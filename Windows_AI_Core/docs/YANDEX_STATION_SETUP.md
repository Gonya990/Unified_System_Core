# Интеграция Яндекс Станция Дуо Макс с Home Assistant

## 🎯 Цель

Полная локальная интеграция Яндекс Станции Дуо Макс (YNDX-00055) с Home Assistant для:

- 🔊 Управления воспроизведением (TTS, медиа)
- 🏠 Контроля умного дома через Алису
- 📡 Использования встроенного Zigbee хаба
- 🎤 Голосовых команд для устройств HA

---

## 📋 Предварительные требования

1. **Home Assistant** установлен и работает
2. **HACS** (Home Assistant Community Store) установлен
3. **Яндекс Станция** подключена к той же сети
4. **Аккаунт Яндекс** с привязанной станцией

---

## 🚀 Установка через HACS

### Шаг 1: Добавление репозитория

1. Откройте **HACS** в боковом меню Home Assistant
2. Нажмите на **три точки** (⋮) в правом верхнем углу
3. Выберите **"Custom repositories"** (Пользовательские репозитории)
4. Введите:
   - **Repository**: `AlexxIT/YandexStation`
   - **Category**: `Integration`
5. Нажмите **"ADD"**

### Шаг 2: Установка интеграции

1. В HACS найдите **"Yandex.Station"**
2. Нажмите **"INSTALL THIS REPOSITORY IN HACS"**
3. Выберите **последнюю версию**
4. Нажмите **"INSTALL"**
5. **Перезагрузите Home Assistant**:
   - Настройки → Система → Перезагрузить

### Шаг 3: Настройка интеграции

1. Перейдите в **Настройки** → **Устройства и службы** → **Интеграции**
2. Нажмите **"+ Добавить интеграцию"**
3. Найдите **"Yandex.Station"**
4. Выберите метод авторизации:

#### Метод 1: QR-код (Рекомендуется)

1. Отсканируйте QR-код приложением **Яндекс** на телефоне
2. Подтвердите вход
3. Готово!

#### Метод 2: Cookies

1. Откройте <https://yandex.ru> в браузере
2. Войдите в аккаунт
3. Откройте DevTools (F12) → Application → Cookies
4. Скопируйте значение cookie `Session_id`
5. Вставьте в поле интеграции

#### Метод 3: Токен

1. Получите токен через <https://oauth.yandex.ru>
2. Вставьте в поле интеграции

## 🎛 Настройка устройств

После успешной авторизации:

1. **Яндекс Станция** появится как **Media Player** в HA
2. Все **Zigbee устройства** станции будут доступны
3. Можно управлять через:
   - Lovelace UI
   - Автоматизации
   - Скрипты
   - Наш AI Bot (`/ha` команды)

---

## 🔧 Использование

### TTS (Text-to-Speech)

```yaml
service: tts.yandex_station_say
data:
  entity_id: media_player.yandex_station_duo_max
  message: "Привет! Это тест голосового сообщения"
```

### Управление воспроизведением

```yaml
service: media_player.play_media
data:
  entity_id: media_player.yandex_station_duo_max
  media_content_type: music
  media_content_id: "Imagine Dragons"
```

### Zigbee устройства

Все Zigbee устройства, подключенные к станции, автоматически появятся в HA как отдельные entities (light.*, switch.*, sensor.*).

## 🤖 Интеграция с AI Bot

После настройки можно управлять станцией через наш бот:

```
/ha status - проверить подключение
/ha sensors - показать датчики (включая от Zigbee)
```

Также станция будет доступна через HomeKit Bridge:

```
/homekit start
```

## 🎤 Голосовые команды Алисе

Теперь Алиса на станции может управлять устройствами из Home Assistant:

- "Алиса, включи свет в гостиной" (если свет добавлен в HA)
- "Алиса, какая температура дома?" (если есть датчики)
- "Алиса, запусти сценарий доброе утро" (если создан script в HA)

Для этого нужно дополнительно установить **Yandex Smart Home** интеграцию:

1. HACS → Интеграции → Найти "Yandex Smart Home"
2. Установить
3. Настроить в приложении Яндекс (Устройства → Умный дом → Добавить → Home Assistant)

## 🔍 Troubleshooting

### Станция не обнаружена

- Убедитесь, что станция и HA в одной сети
- Проверьте, что станция включена и подключена к интернету
- Перезагрузите интеграцию

### Ошибка авторизации

- Попробуйте другой метод (QR-код обычно работает лучше)
- Проверьте, что аккаунт Яндекс активен
- Очистите cookies браузера и попробуйте снова

### Zigbee устройства не появляются

- Убедитесь, что устройства подключены к станции (проверьте в приложении Яндекс)
- Перезагрузите станцию
- Обновите интеграцию до последней версии
